## üíæ How to Add Downloads to Course

### Option 1: Direct File Links
In your course lesson, add:

```
Download: [GMB_POST_IDEAS.md](https://yourdomain.com/course-assets/GMB_POST_IDEAS.md)
```

### Option 2: Embedded Content
Copy the content directly into the lesson content area (for easier reading in-browser)

### Option 3: PDF Version
Convert each .md file to PDF first, then upload

**Recommended:** Option 1 (direct links) - easiest to maintain and update
